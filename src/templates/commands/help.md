# Command: /stdd:help

## Usage
```
chaos help overview                          # Show STDD overview
chaos help workflow                          # Show workflow guides
chaos help list                              # List all commands
chaos help search "<term>"                   # Search commands
chaos help context                           # Show context-specific help
chaos help troubleshoot                      # Show troubleshooting guide
chaos help cheatsheet                        # Show quick reference
```

## Description
Provide context-aware help based on current project state and user needs, including workflows, troubleshooting, and command reference.

## Execution Flow
1. Detect current project state
2. Identify relevant commands
3. Generate contextual help
4. Provide troubleshooting guidance
5. Show next step recommendations

## Output
- Context-aware help text
- Command reference
- Workflow guides
- Troubleshooting suggestions
