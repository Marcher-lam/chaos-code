# Command: /stdd:baby-steps

## Usage
```
chaos baby-steps [task]           # Guide through TDD in small steps
chaos baby-steps --auto           # Auto-advance mode
```

## Description
Interactive TDD guidance for complex components, breaking implementation into small, manageable steps.

## Execution Flow
1. Parse task description
2. Break down into micro-steps
3. Guide through Red-Green-Refactor cycle
4. Provide feedback at each step

## Output
- Step-by-step TDD guidance
- Progress tracking
- Quality checkpoints
