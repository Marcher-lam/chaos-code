# Command: /stdd:memory-scan

## Usage
```
stdd memory-scan                 # Scan memory
stdd memory scan                 # Alias
stdd memory search <query>       # Semantic search
stdd memory stats                # Memory statistics
```

## Description
Scans and searches vector database memory for semantic pattern matching.

## Execution Flow
1. Load memory index
2. Perform semantic search
3. Rank results by relevance
4. Display matches

## Output
- Search results
- Similarity scores
- Memory statistics
