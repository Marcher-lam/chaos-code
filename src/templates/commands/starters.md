# Command: /stdd:starters

## Usage
```
stdd starters list               # List templates
stdd starters create <type>      # Create from template
```

## Description
Manages project starter templates for TypeScript, JavaScript, Python, Go, and Rust.

## Execution Flow
1. List available templates
2. Select template type
3. Copy template to project
4. Initialize configuration

## Output
- Template list
- Initialized project
