# Command: /stdd:start

## Usage
```
stdd start                       # Launch wizard
```

## Description
Interactive quick start wizard (TTY) or help text (non-TTY) for new users.

## Execution Flow
1. Check TTY availability
2. Display interactive wizard or help
3. Guide through initialization
4. Provide next steps

## Output
- Interactive prompts
- Help text
