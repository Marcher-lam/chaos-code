# Command: /stdd:roles

## Usage
```
chaos roles                       # List roles
chaos roles <action>              # Use role
chaos roles --mode=party          # Party Mode debate
```

## Description
12 professional Agent roles (4 basic + 8 specialized) with Party Mode debate and adversarial security review.

## Execution Flow
1. Display available roles
2. Select role for task
3. Execute role-specific workflow
4. Capture role output

## Output
- Role descriptions
- Task execution by role
- Debate transcripts
